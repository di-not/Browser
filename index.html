Web Lifecycle Map

База знаний для FigJam-графа о том, как браузер проходит путь от ввода адреса до загрузки ресурса и обработки ответа.

Страницы графа

  • Ответ из Service Worker / Cache Storage / fallback в сеть
  • Ответ из Memory/Disk HTTP Cache
  • Локального ответа нет → DNS → соединение
  • HTTP Request — запрос
  • HTTP Response — ответ
  • CSS loading и CSSOM
  • DOMContentLoaded → subresources → load
  • Style → Layout → Paint → Composite
  • Static HTML-документ
